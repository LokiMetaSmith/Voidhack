# Star Trek Sound Effects

To enable authentic Star Trek LCARS computer sounds, place audio files in this directory with the following names:

## Required Sound Files

1. **activate.mp3** - Played when activating voice input (communicator button press)
2. **listening.mp3** - Played when actively listening to user speech
3. **processing.mp3** - Played while processing AI response
4. **complete.mp3** - Played when computer finishes speaking response
5. **error.mp3** - Played when an error occurs
6. **button.mp3** - Generic button press sound (for UI interactions)

## Where to Get Star Trek Sounds

### Recommended Sources:
- **TrekCore Audio Library**: https://www.trekcore.com/audio/
- **MediaCollege TNG Sounds**: https://www.mediacollege.com/downloads/sound-effects/star-trek/tng/
- **Star Trek Dimension**: https://www.stdimension.org/MediaLib/computere.htm

### Suggested Sounds:
- **Activate**: LCARS panel beep or chirp
- **Listening**: Computer processing tone
- **Processing**: Data processing sound
- **Complete**: Affirmative beep or completion chime
- **Error**: Alert or warning tone
- **Button**: Generic LCARS button press

## File Format
- **Format**: MP3 (preferred) or WAV
- **Quality**: 128kbps or higher
- **Duration**: Keep sounds short (0.5-2 seconds)

## Usage Note
The application will work without these sound files - they are optional enhancements. If files are not present, the app will continue to function normally without sound effects.
